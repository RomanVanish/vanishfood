# Сайт Андрея Ванишевского — как запустить локально

## Что внутри
- `index.html` — сам сайт (откройте этот файл)
- `styles.css` — оформление
- `app.js` — анимации, параллакс, формы
- `image-slot.js` — рамки под фото (drag-and-drop)
- `tweaks-panel.jsx`, `tweaks.jsx` — панель быстрых настроек (нужна только в режиме редактирования)
- `Превью на телефоне.html` — витрина мобильной версии в рамке телефона

## Способ 1 — просто открыть (быстро)
Дважды кликните по `index.html` — сайт откроется в браузере.
Нужен интернет: шрифты подгружаются из сети.

## Способ 2 — через локальный сервер (рекомендую)
Так работает абсолютно всё, без ограничений браузера.

Откройте терминал/командную строку в этой папке и выполните одну из команд:

**Если установлен Python (есть почти везде):**
```
python -m http.server 8080
```
Затем откройте в браузере: http://localhost:8080

**Если установлен Node.js:**
```
npx serve
```
и откройте адрес, который он покажет.

## Как вставить свои фото
Откройте сайт, найдите серые рамки с подписью «перетащите фото» и перетащите туда изображение прямо мышкой — оно сохранится в браузере.
